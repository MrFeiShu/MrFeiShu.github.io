# ABC线

ABC三条线路是指从天门山收票站和天门山景区的交通路线。

线路|上山|下山
-|-|-
A线|天门山索道|穿山扶梯--->天门山快线索道--->接驳班车
B线|接驳班车--->天门山快线索道--->穿山扶梯|天门山索道
C线|接驳班车--->天门山快线索道--->穿山扶梯|穿山扶梯--->天门山快线索道--->接驳班车

# 东西线

东西线是指天门山景区内的游览路线。

天门山景区的游览路线可以看成一个环形路线，被**天门山索道上站**和**天门山寺**分为两段。

东边的即为**东线**；西边的即为**西线**。
